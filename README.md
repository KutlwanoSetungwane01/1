# To-Do List App

To-Do List application** written in Python. 

## Features
- Add new tasks  
- View all tasks  
- Remove completed tasks  
- Saves tasks to a file (`tasks.txt`)  

## How to Run
-Downloaded  the recomended python extension to run the program
